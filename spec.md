objetivo
Quiero una pagina web con las siguientes caracteristicas:
- diseño moderno
- responsive
- interactiva
esta aplicacion es para estudiantes del sena donde puedan consultar documentos de estudios de los diferentes programas que ofrece el sena y tambien puedan ver las ofertas de empleo que hay para los estudiantes del sena, ademas de diferentes recursos que puedan serles utiles en su vida academica y profesional. la pagina web debe ser capaz de ejecutarce en navegadores web, con las siguientes caracteristicas:
- chrome
- firefox
- safari
- edge
- opera

Esta información también puede ir proporcionandola docentes del sena, por ejemplo si hay algun documento o enlace de interes para los estudiantes, lo puedan compartir por medio de esta plataforma, y a los estudiantes les pueda aparecer en su pagina web como una sugerencia o recomendado.ademas de que el usuario pueda tener un perfil donde pueda ver su progreso en la aplicacion.
